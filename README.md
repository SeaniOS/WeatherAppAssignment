# Weather App Assignment
This is my assignment for a great job opportunity

> [!NOTE]
Project Overview
> 1. This project is created using `Interface: Storyboard` for initial UIKit codebase as per team's project.
> 2. No external libraries have been used. Internal frameworks: UIKit, SwiftUI, Combine, CoreData, XCTest.
> 3. Coverage 100% for all files except `CityView.swift` (SwiftUI view). Please see the evidence at the Coverage section below.
> 4. Architecture: CLEAN architecture (including Presentation-Domain-Data layers) with dependency injection.
> 5. Design Patterns: Singleton, Facade, Repository.
